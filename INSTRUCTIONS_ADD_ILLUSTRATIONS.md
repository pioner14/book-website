# Инструкция по добавлению иллюстраций в главы книги

## Общая информация

В этом руководстве описано, как вручную добавлять иллюстрации в главы книги с помощью компонента `ChapterImage`.

## Структура иллюстраций

Иллюстрации должны быть размещены в папке `/public/chapter-images/[номер_главы]/`:
- `chapter-images/1/` - для иллюстраций к главе 1
- `chapter-images/2/` - для иллюстраций к главе 2
- и так далее

## Как добавить иллюстрации в главу

1. **Подготовьте изображения**:
   - Сохраните изображения в соответствующую папку `/public/chapter-images/[номер_главы]/`
   - Рекомендуется использовать понятные имена файлов (например, `bertha_in_suitcase.jpg`, `phoebe_at_market.jpg`)

2. **Откройте MDX файл главы**:
   - Файлы находятся в `/src/content/book/[язык]/` (например, `/src/content/book/ru/chapter1.mdx`)

3. **Вставьте иллюстрации в подходящие места текста**:
   - Найдите логически подходящее место в тексте
   - Вставьте компонент:

   ```md
   <ChapterImage src="/chapter-images/1/название_вашего_файла.jpg" alt="Описание изображения" />
   ```

4. **Пример корректной вставки**:
   ```md
   # Глава 1: Фиби и чемоданное приключение Берты
   Как-то раз, одним солнечным днём, Фиби пошла на прогулку и встретила свою подругу Берту. Она почему-то была грустная, и Фиби спросила:
   — Почему ты такая грустная?
   Берта сказала, что её семья собирает чемоданы и хочет куда-то уезжать, кажется, в Альпы, но её не берут… А она тоже хочет гоняться за оленями, валяться в травке и собирать цветочки.
   Фиби спросила, почему Берта думает, что её не берут.
   Берта ответила:
   — Потому что мой любимый мячик не упаковали в чемодан.
   <ChapterImage src="/chapter-images/1/bertha_sad.jpg" alt="Берта выглядит грустной" />
   Фиби ужаснулась и поняла: сомнений нет — Берту оставляют одну...
   ```

## Рекомендации по качеству

1. **Атрибут `alt`**:
   - Обязательно заполняйте атрибут `alt` описанием изображения
   - Это важно для доступности и SEO
   - Используйте осмысленные описания (например, "Фиби встречает грустную Берту" вместо "изображение 1")

2. **Размещение изображений**:
   - Вставляйте изображения в логически подходящие места текста
   - Не размещайте слишком много изображений подряд
   - Следите за тем, чтобы изображение поддерживало текстовую историю

3. **Размеры изображений**:
   - Рекомендуется оптимизировать изображения для веба
   - Используйте форматы JPEG для фотографий и PNG для изображений с прозрачностью

## Проверка результата

1. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

2. Перейдите на страницу главы в браузере, чтобы убедиться, что изображения корректно отображаются

## Примеры успешных вставок

В главе 1 (`chapter1.mdx`) уже есть примеры корректно вставленных изображений:
```md
<ChapterImage src="/chapter-images/1/97EB435B-AAE5-4CF1-81F2-3A0BC7B93A65.jpeg" alt="Фиби встречает грустную Берту" />
```

## Распространенные ошибки

1. **Забытый атрибут `alt`** - всегда указывайте осмысленное описание
2. **Неправильный путь к изображению** - убедитесь, что путь начинается с `/chapter-images/`
3. **Изображения без контекста** - вставляйте изображения в логически подходящие места текста
4. **Дублирование изображений** - не вставляйте одно и то же изображение дважды

## Примеры для разных языков

Иллюстрации для разных языковых версий одной и той же главы могут отличаться, но должны отражать одинаковые сцены:

- Русская версия: `/src/content/book/ru/chapter1.mdx`
- Украинская версия: `/src/content/book/ua/chapter1.mdx`
- Французская версия: `/src/content/book/fr/chapter1.mdx`

Каждая языковая версия может содержать свои уникальные иллюстрации, соответствующие культурному контексту.
